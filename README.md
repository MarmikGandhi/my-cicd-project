# My CI/CD Project

A simple project demonstrating the implementation of CI/CD (Continuous Integration and Continuous Deployment) using modern DevOps practices.

## Overview

This project showcases how to automate:
- Building the application
- Running tests
- Deploying code automatically

## Tech Stack

- Git & GitHub
- GitHub Actions (CI/CD pipeline)
- JavaScript

## CI/CD Workflow

Triggered on:
- Push to repository
- Pull requests

Steps:
1. Checkout code
2. Install dependencies
3. Run tests
4. Build project
5. Deploy

## Project Structure

my-cicd-project/
│── .github/workflows/
│── src/
│── README.md

## Getting Started

```bash
git clone https://github.com/MarmikGandhi/my-cicd-project.git
cd my-cicd-project
```

Install dependencies:
```bash
npm install
# or
pip install -r requirements.txt
```

Run project:
```bash
npm start
# or
python app.py
```

## Running Tests

```bash
npm test
# or
pytest
```

## Deployment

Automated via GitHub Actions after successful tests.
